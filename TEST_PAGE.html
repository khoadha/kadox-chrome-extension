Debug Information

Open the Chrome DevTools Console (F12) to see debug logs from the extension.

You should see logs starting with [Kadox] when you use the extension.

Check for:

  • [Kadox] Content script loaded - means the content script is injected
  • [Kadox] Text/Image analysis started - means the context menu was clicked
  • [Kadox] Showing loading modal - means the modal should appear
  • Any error messages in red
